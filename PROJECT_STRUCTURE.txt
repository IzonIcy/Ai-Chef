TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

❓ Python not found in app?
   → Edit PythonBridge.swift
   → Change path to: /usr/bin/python3 (or run: which python3)

❓ App can't find recipes?
   → Check recipes.py is in parent directory
   → Test with: python3 app_bridge.py find_recipes '["chicken"]'

❓ Changes in recipes.py don't show?
   → Close app (Cmd+Q) and reopen

❓ OpenAI not working?
   → Check .env file exists
   → Has OPENAI_API_KEY=sk-...
   → Restart app after adding
